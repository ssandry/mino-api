# API for the MINO repository.

# Dependencies

- Flask==1.1.2
- gunicorn==20.0.4
- itsdangerous==1.1.0
- Jinja2==2.11.2
- MarkupSafe==1.1.1
- Werkzeug==1.0.1

# Setup

**Install project**

```sh
$git clone https://github.com/ssandry/mino-api.git
$cd mino-api
```

# Start script

```sh
$python3 app.py
```
