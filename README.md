### REST API for the MINO repository.

**Live preview [API].**
**Deployed thanks to [Heroku].** <br>

## Setup

**Install project**

```sh
$git clone https://github.com/ssandry/mino-api.git
$cd mino-api
```

## Start script

```sh
$python3 app.py
```
