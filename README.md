# API for the MINO repository.

# Dependencies

<li>Flask==1.1.2</li>
<li>gunicorn==20.0.4</li>
<li>itsdangerous==1.1.0</li>
<li>Jinja2==2.11.2</li>

# Setup

**Install && setup project**

```sh
$git clone https://github.com/ssandry/mino-api.git
$cd mino-api
```

# Start script

```sh
$python3 app.py
```
